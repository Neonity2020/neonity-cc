neonity.cc `0.1.0` 是站点内容系统的第一个稳定版本。从这个版本开始，技术文章与产品发布记录将分别维护，拥有独立的内容模型和访问路径。

## 新增

- 基于 Astro Content Collections 的文章管理能力
- 独立的 Release 列表与详情页面
- 中文标签自动聚合
- 文章封面、作者、发布日期与草稿状态
- 面向桌面和移动设备的响应式阅读体验

## 改进

- 将站点名称和 canonical URL 统一为 `neonity.cc`
- 优化文章正文的排版、间距与暗色模式表现
- 为内容作者补充 frontmatter 示例和发布说明

## 已知限制

- 当前版本尚未提供 RSS 订阅
- Release 暂不支持按状态筛选

后续版本会继续完善订阅、搜索与内容发现能力。
