今天，我们正式发布 **neonity.cc 1.0**。

neonity.cc 是一个专注于 AI 技术重要思考的独立内容平台。我们关注的不只是模型参数、排行榜和短期热点，更关心技术如何成为可靠的产品，如何改变人与软件的关系，以及这些变化将带来什么长期影响。

1.0 标志着站点内容基础设施已经稳定，可以支持持续写作与版本演进。

## 文章体系

- 使用 Markdown 和 MDX 编写内容
- 支持标题、摘要、作者、发布日期与更新时间
- 支持文章封面和无障碍图片说明
- 通过 `draft` 控制文章是否公开
- 按发布日期自动排序

## 内容发现

- 自动聚合文章标签
- 提供独立标签索引
- 为每个标签生成文章列表页
- 首页展示最新文章与核心内容定位

## 独立 Release 体系

文章与 Release 不再共享内容模型。Release 拥有独立的版本号、发布状态、版本亮点和破坏性变更标记，并通过专属时间线展示产品演进。

支持的发布状态包括：

- `stable`：稳定版本
- `beta`：公开测试版本
- `alpha`：早期预览版本

## 阅读体验

- 面向桌面与移动设备的响应式布局
- 自动适配系统明暗主题
- 优化中文正文排版、代码样式和内容层级
- 为文章、标签与 Release 提供独立 SEO 信息
- canonical URL 统一使用 `https://neonity.cc`

## 1.0 的内容方向

neonity.cc 将持续关注以下主题：

1. 基础模型与 AI 产品的关键进展
2. Agent、上下文工程与可靠性实践
3. 人机交互、组织方式与社会影响
4. AI 技术背后的长期趋势与重要判断

## 已知限制

- 暂未提供全文搜索
- 暂未提供 RSS 或邮件订阅
- Release 暂不支持按状态筛选

这些能力会在后续版本中逐步完善。1.0 不是终点，而是 neonity.cc 开始持续记录、思考和发布的起点。
