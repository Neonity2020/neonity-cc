## What belongs here

This documentation area is the durable memory for NEONITY.cc. It collects the rules and context that should survive beyond a single page implementation.

- **System notes** explain how the site is organized.
- **Design notes** keep the visual language reusable.
- **Authoring notes** make it easier to add pages without re-learning the project.
